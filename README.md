# aws-nginx-customhtmlwebpage
# AWS Nginx Web Server on EC2

## Project Overview
This project demonstrates how to host a custom HTML webpage on an AWS EC2 Ubuntu instance using the Nginx web server.

## Technologies Used
- AWS EC2
- Ubuntu
- Nginx
- HTML

## Steps
1. Launched an Ubuntu EC2 instance.
2. Connected using EC2 Instance Connect.
3. Installed Nginx.
4. Created a custom index.html page.
5. Restarted the Nginx service.
6. Accessed the webpage using the EC2 public IP.

## Output
The custom webpage was successfully displayed in the browser.
